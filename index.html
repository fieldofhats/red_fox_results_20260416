1 Occupancy Model Report

1.1 Overview

An occupancy model with detection and temporal structure can be written as:

\[ y_{i,r} \mid z_i \sim \text{Bernoulli}(p_{i,r} \cdot z_i) \]

\[ z_i \sim \text{Bernoulli}(\psi_i) \]

\[ \text{logit}(p_{i,r}) = \alpha_0 + \alpha_1 \cdot \text{lureDays}_{i,r} + s(\text{JD}_{i,r}) \]

\[ \text{logit}(\psi_i) = \beta_0 + \sum_{k} \beta_k \cdot X_{i,k} \]

where the occupancy covariates include:

\[ X_{i,k} \in \{\text{year effects},\; \text{environmental covariates on occupancy}\} \]

1.1.1 Temporal random effect

The detection model includes a smooth seasonal random effect with a snow-dependent interaction:

\[ s(\text{JD}_{i,r}) = \mu_{\text{base}}(\text{JD}_{i,r}) + \text{snow}_i \cdot \mu_{\text{snow}}(\text{JD}_{i,r}) \]

where each component is represented using spline basis functions:

\[ \mu_{\text{base}}(\text{JD}) = \sum_{j=1}^{K} \gamma_j \, B_j(\text{JD}) \]

\[ \mu_{\text{snow}}(\text{JD}) = \sum_{j=1}^{K} \delta_j \, B_j(\text{JD}) \]

with coefficients:

\[ \gamma_j \sim \mathcal{N}(0, \sigma_{\text{base}}^2), \quad \delta_j \sim \mathcal{N}(0, \sigma_{\text{snow}}^2) \]

where:

  • \(B_j(\cdot)\) are spline basis functions (e.g., B-splines)
  • \(\mu_{\text{base}}(\text{JD})\) represents the baseline seasonal detection pattern
  • \(\mu_{\text{snow}}(\text{JD})\) represents how the effect of snow varies across the season
  • \(\sigma_{\text{base}}^2\) and \(\sigma_{\text{snow}}^2\) control the smoothness of each component

The resulting random effect allows detection probability to vary smoothly over time while permitting the influence of snow depth to change across the seasonal cycle.

1.1.2 Variable definitions

  • \(y_{i,r}\): detection/non-detection at site \(i\), replicate \(r\)
  • \(z_i\): latent occupancy state at site \(i\)
  • \(p_{i,r}\): detection probability
  • \(\psi_i\): occupancy probability

1.1.3 Interpretation

This is a hierarchical occupancy model with imperfect detection. Detection probability varies as a function of lure aging and a smooth seasonal effect, modeled using a spline-based random effect. Occupancy probability is modeled as a function of annual variation and environmental covariates, allowing flexible ecological inference.

1.1.4 Base NIMBLE model code

NoteBase occupancy model (NIMBLE) 
code_base_cov_model_snow_spline_re_noncenter <- nimbleCode({
  ## fixed effects
  for(l in 1:n.beta){
    beta[l] ~ dnorm(0, sd = 1)
  }

  for(l in 1:n.alpha){
    alpha[l] ~ dnorm(0, sd = 1)
  }

  ## base JD smooth: non-centered RW2
  sd_beta ~ T(dnorm(0, sd = 1), 0, )
  beta_spline_1 ~ dnorm(0, sd = 1)
  beta_spline_2 ~ dnorm(0, sd = 1)

  beta_spline[1] <- beta_spline_1
  beta_spline[2] <- beta_spline_2

  for(k in 3:K){
    beta_spline_raw[k-2] ~ dnorm(0, sd = 1)
    beta_spline[k] <- 2 * beta_spline[k-1] - beta_spline[k-2] + sd_beta * beta_spline_raw[k-2]
  }

  ## snow-varying JD smooth: non-centered RW2
  sd_beta_s ~ T(dnorm(0, sd = 1), 0, )
  beta_snow_1 ~ dnorm(0, sd = 1)
  beta_snow_2 ~ dnorm(0, sd = 1)

  beta_snow[1] <- beta_snow_1
  beta_snow[2] <- beta_snow_2

  for(k in 3:K){
    beta_snow_raw[k-2] ~ dnorm(0, sd = 1)
    beta_snow[k] <- 2 * beta_snow[k-1] - beta_snow[k-2] + sd_beta_s * beta_snow_raw[k-2]
  }

  ## random effect scale
  sd_s ~ T(dnorm(0, sd = 1), 0, )

  ## precompute smooths on JD grid
  for(d in 1:n_jd){
    mu_base[d] <- inprod(B[d, 1:K], beta_spline[1:K])
    mu_snow[d] <- inprod(B[d, 1:K], beta_snow[1:K])
  }

  ## occupancy / detection model
  for(j in 1:J) {
    z[j] ~ dbern(psi[j])
    logit(psi[j]) <- (beta[1:n.beta] %*% x_occ[j, 1:n.beta])[1,1]

    for(k in 1:nReps[j]) {
      mu_s[rep_index[j,k]] <-
        mu_base[jd_index[rep_index[j,k]]] +
        jd_snow_season[jd_index[rep_index[j,k]]] *
        (snow[j] * mu_snow[jd_index[rep_index[j,k]]])

      eps[rep_index[j,k]] ~ dnorm(0, 1)
      s[rep_index[j,k]] <- mu_s[rep_index[j,k]] + sd_s * eps[rep_index[j,k]]

      p_eff[rep_index[j, k]] <- z[j] * p[rep_index[j, k]]
      y[rep_index[j, k]] ~ dbern(p_eff[rep_index[j, k]])

      logit(p[rep_index[j, k]]) <-
        (alpha[1:n.alpha] %*% x_det[rep_index[j, k], 1:n.alpha])[1,1] +
        s[rep_index[j, k]]
    }
  }
})

1.1.5 Reversible-jump NIMBLE model code

NoteReversible-jump occupancy model (NIMBLE) 
code_base_rev_re_spline <- nimbleCode({
  ## priors for occupancy coefficients
  for(l in 1:n.beta){
    beta[l] ~ dnorm(0, 0.1)
  }

  beta.psi.indicator ~ dbeta(1, 1)
  for(l in 1:n.beta.rev){
    beta.switch[l] ~ dbern(beta.psi.indicator)
    beta.indicator[l] <- beta.switch[l] * beta[l + n.beta.no]
  }

  ## priors for detection coefficients
  for(l in 1:n.alpha){
    alpha[l] ~ dnorm(0, 0.1)
  }

  alpha.psi.indicator ~ dbeta(1, 1)

  ## combine intercept and detection covariates in one matrix
  alpha.indicator[1] <- alpha[1]
  if(n.alpha == 2){
    alpha.switch ~ dbern(alpha.psi.indicator)
    alpha.indicator[2] <- alpha.switch * alpha[2]
  } else {
    for(l in 2:n.alpha){
      alpha.switch[l-1] ~ dbern(alpha.psi.indicator)
      alpha.indicator[l] <- alpha.switch[l-1] * alpha[l]
    }
  }

  ## base JD smooth: RW2 prior on beta_spline[1:K]
  beta_spline[1] ~ dnorm(0, sd = 1)
  beta_spline[2] ~ dnorm(0, sd = 1)
  for(k in 3:K){
    mu_k[k-2] <- 2 * beta_spline[k-1] - beta_spline[k-2]
    beta_spline[k] ~ dnorm(mu_k[k-2], tau_beta)
  }
  sd_beta ~ dunif(0, 2)
  tau_beta <- 1 / (sd_beta * sd_beta)

  ## snow-varying JD smooth: RW2 prior on beta_snow[1:K]
  beta_snow[1] ~ dnorm(0, sd = 1)
  beta_snow[2] ~ dnorm(0, sd = 1)
  for(k in 3:K){
    mu_k_s[k-2] <- 2 * beta_snow[k-1] - beta_snow[k-2]
    beta_snow[k] ~ dnorm(mu_k_s[k-2], tau_beta_s)
  }
  sd_beta_s ~ dunif(0, 2)
  tau_beta_s <- 1 / (sd_beta_s * sd_beta_s)

  ## random effect scale
  sd_s ~ dunif(0, 2)

  ## precompute smooths on JD grid
  for(d in 1:n_jd){
    mu_base[d] <- inprod(B[d, 1:K], beta_spline[1:K])
    mu_snow[d] <- inprod(B[d, 1:K], beta_snow[1:K])
  }

  ## occupancy / detection model
  for(j in 1:J) {
    z[j] ~ dbern(psi[j])
    logit(psi[j]) <-
      (beta[1:n.beta.no] %*% x_occ_no[j, 1:n.beta.no])[1,1] +
      (beta.indicator[1:n.beta.rev] %*% x_occ_rev[j, 1:n.beta.rev])[1,1]

    for(k in 1:nReps[j]) {
      mu_s[rep_index[j,k]] <-
        mu_base[jd_index[rep_index[j,k]]] +
        jd_snow_season[jd_index[rep_index[j,k]]] *
        (snow[j] * mu_snow[jd_index[rep_index[j,k]]])

      eps[rep_index[j,k]] ~ dnorm(0, 1)
      s[rep_index[j,k]] <- mu_s[rep_index[j,k]] + sd_s * eps[rep_index[j,k]]

      p_eff[rep_index[j, k]] <- z[j] * p[rep_index[j, k]]
      y[rep_index[j, k]] ~ dbern(p_eff[rep_index[j, k]])

      logit(p[rep_index[j, k]]) <-
        (alpha.indicator[1:n.alpha] %*% x_det[rep_index[j, k], 1:n.alpha])[1,1] +
        s[rep_index[j,k]]
    }
  }
})
